# Privacy Policy for [KAIT.hk]

Last Updated: [2026-07-19]

This Privacy Policy describes how this personal automation tool and application (the "Service") handles data. 

Because this is a personal, non-commercial automation project, the Service is designed strictly for individual use and does not collect or share any third-party user data.

## 1. Data Collection and Usage
- **No Personal Data Collection:** The Service does not collect, store, or monitor any personal identifiable information (PII) from LinkedIn users or the general public.
- **Authentication Tokens:** Any API tokens, access keys, or credentials provided by LinkedIn are stored securely within the owner's private environment (such as GitHub Secrets or local environment variables). These tokens are used solely to authenticate API requests to execute the automation tasks (e.g., posting updates).

## 2. Data Sharing and Disclosure
- **No Third-Party Sharing:** Since no data is collected, no data is ever sold, traded, or shared with third parties.

## 3. Security
We value the security of the automated accounts. All access tokens and configuration settings are stored in secure, private environments to prevent unauthorized access.

## 4. Changes to This Privacy Policy
This policy may be updated from time to time to reflect changes in the automation script. Any updates will be posted directly to this repository.

## 5. Contact
If you have any questions about this Privacy Policy, please contact the repository owner via GitHub / inf@leactgroup.com.
